## このサイトについて

**AI Tech Notes** は、社内ナレッジを対象とした **AI 回答 / ドラフト生成 / レビュー支援** システムを
設計・構築するときに「知っておくべきこと」を、漏れなく一箇所にまとめた技術ノートです。
RAG・MCP・データの蓄積場所（Microsoft 系中心）・データ形式・アンチパターン・コスト/ROI までを
横断的にカバーします。

**想定読者:** AI システムアーキテクト / IT 部門 / 開発者
